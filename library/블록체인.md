## 최신 지식 (2026-07-20)
- **EIP-7702 계정 추상화 본격화**: Pectra(2025.5) 이후 EOA가 스마트 컨트랙트처럼 배치 트랜잭션·가스 스폰서십·소셜 리커버리 지원, 신규 지갑 UX 표준으로 자리잡음 (ethereum.org)
- **Fusaka 업그레이드 반영**: PeerDAS 도입으로 블롭 용량 8배 확대, L1 가스 리밋 60M 표준화(EIP-7935), 트랜잭션당 가스 상한 ~16.78M(EIP-7825) 고려해 대형 멀티콜 설계 필요 (coingecko.com, ethereum.org)
- **ZK 롤업이 사실상 기본값**: 신규 L2는 대부분 ZK 방식 채택, 기존 옵티미스틱 스택(Arbitrum, Optimism)도 ZK 프루프로 전환 중이며 zkSync Era·Scroll·Linea·Starknet이 주요 zkEVM (blog.thirdweb.com, blockchaintechs.io)
- **탈중앙 증명 생성 인프라 부상**: Succinct(PROVE) 등이 "Open Prover Layer"로 ZK 증명 생성을 마켓플레이스화, 롤업·브릿지·AI 에이전트가 증명 생성을 아웃소싱하는 흐름 확산 (blog.thirdweb.com)
- **Foundry가 사실상 표준 툴체인**: Rust 기반 Foundry가 빠른 컴파일·테스트 속도로 기존 JS 기반 툴체인 대체, OpenZeppelin 감사 완료 라이브러리 재사용이 업계 표준 관행으로 정착 (vegavid.com)
- **차기 업그레이드 대비 필요**: 2026년 상반기 Glamsterdam(병렬 실행, enshrined PBS), 하반기 Hegotá(Verkle Trees로 노드 스토리지 절감) 예정 — L1 확장성·탈중앙화 관련 컨트랙트/인프라 설계 시 선제 대응 권장 (ethereum.org, tatum.io)
